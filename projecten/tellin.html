Tellin'

Inleiding

Tellin’ is het eerste bedrijf dat ik ben begonnen samen met 4 andere voor een schoolopdracht. Deze website is bedoeld om klanten geïnteresseerd te raken in jouw diensten. Onze dienst was het maken van een storytelling website. Vandaar de naam Tellin’. Het was erg leuk en interessant om te zien wat goed en wat minder gaat als iedereen andere ideeën heeft. Met Dit bedrijfje hebben we 2 opdrachten binnen kunnen halen.

            if(wScroll > $('.content-wrapper').offset().top - $(window).height()){

Ik heb dit stukje code in de spotlight gezet omdat ik hier trots op ben. Wat het doet? Het zorgt ervoor dat verschillende elementen van de zijkanten van je scherm komen schuiven, wat een leuk interactief gevoel geeft aan de gebruiker. Met dit stuk code heb ik veel geleerd over Javascript waar ik erg blij mee ben en waar ik veel aan heb ik volgende opdrachten.
